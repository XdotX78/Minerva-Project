# Compatibility

This repository should document only public compatibility facts:

- supported operating systems
- supported MCP-capable clients
- known approval/restart steps
- known install limitations

Do not mirror private implementation notes here.

## Current Public Alpha

Installable release assets currently available:

- Linux x86_64
- macOS Apple Silicon
- Windows x86_64

Project direction is wider than that, but public packaging is still being rolled out step by step.
