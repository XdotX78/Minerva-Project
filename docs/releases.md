# Releases

Every public release should include:

- platform archives
- checksums
- minimal install notes
- compatibility notes

Suggested assets:

- `minerva-linux-x86_64.tar.gz`
- `minerva-linux-aarch64.tar.gz`
- `minerva-macos-aarch64.tar.gz`
- `minerva-windows-x86_64.zip`
- `SHA256SUMS`
